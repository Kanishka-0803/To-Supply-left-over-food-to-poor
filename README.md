# To-Supply-left-over-food-to-poor
FoodConnect is a Salesforce-based app that connects food donors with nearby volunteers or NGOs. It automates leftover food collection, tracks deliveries, and generates reports reducing food waste, ensuring hygiene, and efficiently bridging the gap between surplus and hunger.
